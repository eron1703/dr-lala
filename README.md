## Local Development

### Using Python/Flask

```bash
# Install dependencies
pip install flask

# Run the app
python app.py

# Access at http://localhost:8888
```

### Using Docker

```bash
# Build and run with Docker Compose
docker-compose up -d

# Access at http://localhost:8888
```
